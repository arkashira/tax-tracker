# Tax Tracker
A simple tax tracker application.

## Usage
1. Create a new instance of the `TaxTracker` class.
2. Add expense entries using the `add_expense` method.
3. Get expenses by category using the `get_expenses_by_category` method.
4. Save expenses to a JSON file using the `save_to_json` method.
5. Load expenses from a JSON file using the `load_from_json` method.
