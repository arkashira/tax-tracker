# Tax Tracker
A simple income and expense tracking system for freelancers.

## Features
* Add income entries with category and description
* Categorize income entries
* Save and load income entries from JSON file

## Usage
1. Create a `TaxTracker` object
2. Add income entries using `add_income` method
3. Categorize income entries using `categorize_income` method
4. Save income entries to JSON file using `save_to_json` method
5. Load income entries from JSON file using `load_from_json` method
